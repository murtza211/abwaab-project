# 🎓 منصة أبواب التعليمية

موقع ويب تعليمي حديث وجميل مخصص لطلاب منصة أبواب

## 📋 المحتويات

### الملفات الرئيسية:
- **index.html** - الصفحة الرئيسية للموقع
- **style.css** - ملف الأنماط والتصميم
- **script.js** - ملف الوظائف التفاعلية

## ✨ المميزات

### 1. 🎨 واجهة ترحيبية جميلة
- شعار "أبواب" مميز وجذاب
- عنوان مشجع للطلاب
- تأثيرات حركية ناعمة

### 2. 📅 جدول المحاضرات اليومي
- جدول أنيق وسهل القراءة
- إظهار حالة المحاضرات (جاري الآن / قادم)
- تمييز بصري للمحاضرات الجارية
- تأثيرات عند التمرير على الصفوف

### 3. 🎯 أزرار الدروس الملونة
- **الرياضيات** - ملون بألوان دافئة (برتقالي)
- **الفيزياء** - ملون بألوان باردة (أزرق سماوي)
- **اللغة العربية** - ملون بألوان مميزة (بنفسجي)
- تأثيرات تفاعلية عند الضغط والتمرير

### 4. 📝 مساحة ملاحظات اليوم
- منطقة كتابة نصية كبيرة ومريحة
- **حفظ تلقائي** كل 30 ثانية
- زر حفظ يدوي مع إشعارات تأكيد
- زر مسح الملاحظات
- نصائح دراسية مفيدة بجانب المساحة

### 5. 🌈 تصميم عصري وسريع
- ألوان مريحة للعين (أزرق وأبيض)
- تصميم متجاوب يعمل على جميع الأجهزة
- تأثيرات حركية ناعمة
- سرعة تحميل عالية

## 🚀 كيفية الاستخدام

### 1. فتح الموقع
اضغط مزدوجاً على ملف `index.html` أو افتحه في متصفحك المفضل

### 2. المحاضرات اليومية
- تحقق من جدول المحاضرات في الأعلى
- سترى الحالة الحالية للكل محاضرة

### 3. اختيار المادة الدراسية
- اضغط على أحد الأزرار الملونة:
  - 📐 **الرياضيات** - للدروس الرياضية
  - ⚛️ **الفيزياء** - للدروس الفيزيائية
  - 📖 **اللغة العربية** - لدروس اللغة

### 4. كتابة الملاحظات
- اكتب ملاحظاتك في المربع المخصص
- ستُحفظ تلقائياً كل 30 ثانية
- أو اضغط على **💾 حفظ الملاحظات**
- سيتم تخزينها في المتصفح حتى تحذفها

### 5. إدارة الملاحظات
- اضغط **🗑️ مسح** لحذف جميع الملاحظات
- اضغط **💾 حفظ** لحفظ يدوي موثوق

## 💾 حفظ البيانات

الموقع يستخدم **localStorage** في المتصفح لحفظ:
- نص الملاحظات
- تاريخ آخر تعديل

**ملاحظة مهمة:** البيانات تُحفظ محلياً في جهازك فقط ولن تُرسل لأي خادم

## 🔧 التخصيص

### تغيير الألوان
في ملف `style.css`، ابحث عن قسم `:root` وعدّل القيم:

```css
:root {
    --primary-color: #2563eb;  /* اللون الأساسي */
    --math-color: #d97706;    /* لون الرياضيات */
    --physics-color: #0891b2;  /* لون الفيزياء */
    --arabic-color: #7c3aed;   /* لون اللغة العربية */
}
```

### تغيير النصوص
عدّل النصوص في ملف `index.html`:
- عنوان الترحيب
- أسماء المحاضرات
- بيانات الجدول

### إضافة محاضرات جديدة
أضف صفوف جديدة للجدول في `index.html`:

```html
<tr>
    <td>الوقت</td>
    <td>اسم المحاضرة</td>
    <td>اسم المدرس</td>
    <td><span class="badge upcoming-badge">قادم</span></td>
</tr>
```
